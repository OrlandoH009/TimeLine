COBOL — Business Programming

Work on COBOL began around 1960 to create a common business-oriented programming language. COBOL later became widely adopted for corporate data processing and enterprise software.

PROGRAMMING LANGUAGES
1960

DEC PDP-1 — Minicomputer Era Begins

The DEC PDP-1 introduced interactive computing at a lower cost and smaller size than mainframes, enabling new uses such as real-time graphics and fostering developer communities.

MINICOMPUTER
1961

CTSS — Time-Sharing Demonstrated

The Compatible Time-Sharing System (CTSS) demonstrated multi-user interactive computing, paving the way for shared computing resources and early utilities like messaging and editors.

TIME-SHARING
1962

Atlas — Virtual Memory Concept

The Atlas computer introduced virtual memory concepts, allowing systems to treat disk or drum storage as an extension of main memory and enabling more flexible program execution.

MEMORY
1963

Sketchpad — Interactive Graphics

Ivan Sutherland's Sketchpad demonstrated interactive graphical interfaces and object manipulation, foundational ideas for computer graphics and CAD systems.
